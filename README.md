## Description
An auction platform built with NestJS. This application provides the backend for bidders and auctioneers to engage in auction experiences.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ docker-compose up -d

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test:unit

# e2e tests
$ docker-compose up -d
$ npm run test:integration
```
